A structured, practical roadmap to learn HTML from beginner to advanced level with:
- concept notes
- practice tasks
- coding challenges
- mini projects
- interview preparation material

## 🎯 Who this is for
- Beginners starting web development
- Learners revising HTML5 fundamentals
- Developers preparing for frontend interviews
- Mentors/trainers creating HTML learning paths

## 🚀 How to use this repository
1. Start from `01_Beginner_Level/` and complete sections in order.
2. Practice with level-based task files.
3. Solve exercises in `05_Coding_Challenges/`.
4. Build projects from `04_Projects/`.
5. Revise with `06_General_Interview_Questions/`.
